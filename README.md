# Play Framework with Swagger

#### Version information
PlayFramework 2.3.7 with Java8 64Bits.

## Quickstart
Fazer testes com um projeto simples do play e adicionar o swagger.

## Características

* Play2 Puro
* Swagger 2.0
* Utiliza Várias Dependências:

	* jquery
	* bootstrap
	* wagger-play2
